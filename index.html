Chasing the 10 Pound Monster

Work Related

A biggest loser style web app designed specifically for my workplace. The objective is to be the first player to lose 10 pounds. The demo was built with AngularJS, Bootstrap, and Github.

The biggest challenge was learning to use Heroku, Node, and Express for the first time. I had to learn how to write most of my logic on the server and send the finish data via APIs to the client application.

An example is a project requirement that players couldn't see each other weight. An API was use to send certian information based on login or weight entered from the server to be displayed on the client device.
